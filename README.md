🎓 Student Management System
(Angular + Node.js + Express + MongoDB)
📌 Project Overview

The Student Management System is a full-stack web application developed using:

Frontend: Angular

Backend: Node.js with Express.js

Database: MongoDB

This application allows users to:

Register a new account

Login (Session-based authentication, without JWT)

Perform CRUD operations on student records

The project demonstrates complete frontend-backend integration using REST APIs.

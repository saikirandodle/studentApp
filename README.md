# 🎓 Student Management System  
### (Angular + Node.js + Express + MongoDB)

---

## 📌 Project Overview

The **Student Management System** is a full-stack web application developed using:

- **Frontend:** Angular  
- **Backend:** Node.js with Express.js  
- **Database:** MongoDB  

This application allows users to:

- Register a new account  
- Login (Session-based authentication, without JWT)  
- Perform CRUD operations on student records  

The project demonstrates complete frontend-backend integration using REST APIs.

## 🚀 Key Features

### 🔐 Authentication Module
- User Registration
- User Login

- Logout Functionality

### 🎓 Student Management Module
- Add Student
- View Students
- Update Student Details
- Delete Student

---

## 🔄 Application Flow

1. User registers an account.
2. User logs in using credentials.
3. Server creates a session.
4. User accesses dashboard.
5. User performs CRUD operations.
6. Data is stored and retrieved from MongoDB.
